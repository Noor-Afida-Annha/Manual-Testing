## Bug Report Format

Each bug report includes the following information:

- **Bug ID**
- **Title**
- **Steps to Reproduce**
- **Expected Result**
- **Actual Result**
- **Severity** (Low/Medium/High/Critical)
- **Priority** (Low/Medium/High)
- **Status** (Open/Closed)
- **Screenshot** (If applicable)
- **Notes/Suggestions**
